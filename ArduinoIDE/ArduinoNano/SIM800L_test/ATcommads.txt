AT - ответ OK, если всё впорядке

ATI - информация

AT+GSN - IMEI модуля

AT+CCALR? - возможно ли совершать звонки

AT+COPS? - инфо об операторе

AT+CBC - вольтаж

ATD+79518233333; - звонить

ATH - сбросить звонок

ATA - принять звонок

AT+CSQ - запрос качества связи


ATE0V0+CMEE=1;&W - РЕЖИМ production
	ATE0 - Echo Mode выключен
	ATV0 - цифровой код ответов
	AT+CMEE=1 - цифровой код ошибок
	AT&W - сохранение всех заданных параметров
	

ATE1V1+CMEE=2;&W - РЕЖИМ development
	ATE1 - Echo Mode включен
	ATV1 - текстовый формат ответов
	AT+CMEE=2 - текстовый код ошибок
	AT&W - сохранение всех заданных параметров
	

AT+CRSL=<value> - value - громкость звонка от 0 до 100 в %

AT+CLVL=<value> - value - громкость динамика от 0 до 100 в %

ATS0=<value> - приняь звонок после <value> гудков (value - от 0 до 10) 0 - откл

AT+CLIP=<value> - определитель номера (value - 0 или 1)


AT&W - сохранение всех заданных параметров


AT+CMGF=1 - текстовый формат SMS

AT+CMGS="+79518233333" - отправка SMS, в конце необходим символ "ctrl + Z"

"" - символ "ctrl + Z"
