Список изменений Standard Peripherals Library.

--------------------------------------------------------------------------------

Об архиве Standard Peripherals Library:

В архиве SPL собраны соответсвующие SPL для различных МК. Полный список изменений необходимо смотреть в changelog файле на соответсвующий МК.

--------------------------------------------------------------------------------

v1.0 28/07/2022

    - Первоначальная версия

Версии библиотек для МК в составе SPL:

    - SPL_MDR32FxQI   - Библиотека для МК 1986ВЕ92QI,К1986ВЕ1QI,К1901ВЦ1QI - версия 1.0
    - SPL_MDR1986VK01 - Библиотека для МК К1986ВК01GI                      - версия 1.3
    - SPL_MDR32F02    - Библиотека для МК К1986ВК025                       - версия 0.31
