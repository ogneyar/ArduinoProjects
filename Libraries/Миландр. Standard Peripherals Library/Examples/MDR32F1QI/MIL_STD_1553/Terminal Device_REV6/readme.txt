@par Example Description

This example is intended to show the functionality of microcontroller MDR32F1QI
in TERMINAL mode of the standart GOST R 52070-2003.

This code write to microcontroller MDR32F1QI rev. 6. In other
revision of MCU this code may performed with distinctions.


@par Directory contains:

    - main.c               Main program


@par Hardware and Software environment:

    - This example is intended to run on MDR32F1QI eval board with MDR32F1QI microcontroller.


@par How to use.

To launch the example, you must do the following:
    - Create a project and setup all project configurations.
    - Add main.c file.
    - Edit the MDR32FxQI_config.h to set appropriate run-time parameter checking level.
    - Compile and link together all .c files and load your image into the target board.
    - Run the example.
